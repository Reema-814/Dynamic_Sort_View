Bubble Sort

  • Comparison-Based Algorithm: Bubble Sort repeatedly steps through the list, compares adjacent elements, and swaps them if they are in the wrong order.
  • Inefficient for Large Datasets: It has a poor average and worst-case time complexity of O(n^2), making it unsuitable for large datasets.
  • Stable Sorting: Bubble Sort is stable, meaning it preserves the relative order of equal elements.
  • Simple and Easy to Implement: Despite its inefficiency for large datasets, Bubble Sort is straightforward and often used for educational purposes.
  • Named for "Bubbling" Effect: The algorithm's name comes from the way larger elements "bubble" to the top of the list with each pass.

How Bubble Sort Works

  1. Start at the Beginning: Begin at the start of the list.
  2. Compare Adjacent Elements: Compare the first two elements.
  3. Swap if Necessary: If the first element is greater than the second, swap them.
  4. Move to the Next Pair: Move to the next pair of elements and repeat the comparison and swap process.
  5. Repeat: Continue this process until the end of the list is reached. At this point, the largest element will have "bubbled" to the end of the list.
  6. Iterate: Repeat the entire process for the remaining elements (excluding the last element which is now sorted) until no more swaps are needed.

Time Complexity

  • Worst Case: O(n^2)
  • Average Case: O(n^2)
  • Best Case: O(n)

Space Complexity

O(1) - Bubble sort is an in-place sorting algorithm.

Advantages

  • Simple to understand and implement.
  • Stable sorting algorithm.
  • Requires only a constant amount of additional memory space.
  • Effective for small data sets or nearly sorted lists.

Disadvantages

  • Inefficient for large data sets due to its O(n^2) time complexity.
  • Slower compared to more advanced algorithms like quicksort and mergesort.
  • Makes many unnecessary comparisons even after the list is sorted.

Applications

  • Educational purposes
  • Small datasets
  • Partially sorted arrays
  • Check if the array is sorted
  • Embedded systems
